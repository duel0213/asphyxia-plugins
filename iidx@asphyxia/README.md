# beatmaniaIIDX

Plugin Version: **v0.1.4**

---

Supported Versions

  - beatmaniaIIDX 19 Lincle
  - beatmaniaIIDX 27 HEROIC VERSE
  - beatmaniaIIDX 28 BISTROVER
  - beatmaniaIIDX 29 CastHour
  - beatmaniaIIDX 30 RESIDENT

---

Features

  - STEP UP (Partial)
  - SKILL ANALYZER
  - EVENT
  - ARENA (Local)
  - RANDOME LANE TICKET
  - SONG SELECTION NOTES
